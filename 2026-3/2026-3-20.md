#### .each_char
- 文字列を１文字ずつ処理する
- .eachメソッドは**配列**や**ハッシュ**に対して使用する。

#### 今日のコード
```
n, m = gets.split(' ').map(&:to_i) # n = 店舗数, m = 営業期間(月)
a, b, c = gets.split(' ').map(&:to_i) # a=建築費, b=人件費(月), c=ラーメン単価
close_stores = 0

n.times do
   sold_count = gets.to_i # ラーメンの販売数取得
   profit = c * sold_count - a - m * b
       close_stores += 1 if profit < 0
end

puts close_stores
```
